Проверка Юридических лиц. Реализовать консольное приложение выполняющее запрос к API http://npchk.nalog.ru/
На вход приложению подается ИНН ЮЛ/ФЛ приложение возвращает один из возможных вариантов признака состояния, описанных в документации (ws2.doc).
Датой сделки считать текущую дату.
Приложение использует библиотеку JAX-WS. 
Для работы с JDK9 требуется в VM-options прописать -ea --add-modules=java.se.ee
